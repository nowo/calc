# @wzo/calc

`@wzo/calc` 是一个 **精度数学 + 数字格式化** 库， 支持表达式求值，**0 运行时依赖**。

## 为什么需要这个库

JavaScript 原生浮点数有精度问题：

```js
0.1 + 0.2 // 0.30000000000000004
0.3 - 0.1 // 0.19999999999999998
0.1 * 0.2 // 0.020000000000000004
Number(1.005).toFixed(2) // "1.00"（应该是 "1.01"）
```

`@wzo/calc` 给出严格的十进制结果：

```ts twoslash
import { calc } from '@wzo/calc'
// ---cut---
calc('0.1 + 0.2') // "0.3"
calc('0.3 - 0.1') // "0.2"
calc('0.1 * 0.2') // "0.02"
calc('1.005', { _fmt: { decimals: 2, rounding: 'round' } }) // "1.01"
```

## 核心特性

- **精度算术** —— 内部用 BigInt 表示「digits × 10^(-exp)」，所有有理运算严格正确
- **富格式化** —— 千分位、百分号、压缩、分数、科学记数法、欧式 / 印度风格、4 种舍入策略，类型友好的对象式配置
- **链式 API** —— `chainAdd(10).sub(3).mul(2)()`
- **聚合** —— `calcSum / calcAvg / calcMax / calcMin`
- **0 运行时依赖** —— ~12KB gzip，浏览器 / Node 都能跑
- **完整 TypeScript** —— 类型推导跟手
