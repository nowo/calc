# 数字格式化

格式化通过一个 **`IFormat` 对象**配置，三处都接受它：

- `fmt(value, format)` —— 直接格式化
- `calc(expr, { ..., _fmt })` —— 表达式结果格式化
- 链式终止符 `chainAdd(...)(format)`

对象的每个字段都有完整类型提示和补全，写错即报错。

```ts twoslash
import { calc, chainAdd, fmt } from '@wzo/calc'
// ---cut---
fmt(1234.5, { decimals: 2, thousands: true }) // "1,234.50"
calc('9.9 * 3', { _fmt: { decimals: 2 } }) // "29.70"
chainAdd(1000, 2000)({ decimals: 2, thousands: 'eu' }) // "3.000,00"
```

## 完整字段表
| 字段 | 类型 | 作用 |
| :--- | :--- | :--- |
| `decimals` | `number \| { min?, max? }` | 小数位数（默认保留原始位数）：<br>• 数字 = **固定** N 位，不足补 0（`2`→`"1.00"`）<br>• `{ max }` = 最多 N 位，多余截掉、不补零<br>• `{ min }` = 最少 N 位，不足补零 |
| `rounding` | `'truncate' \| 'halfUp' \| 'banker' \| 'ceil' \| 'floor' \| 'expand'`<br>` 'trunc' \| 'round' \| 'halfEven'` | 配合 `decimals` 取舍，**默认 `'truncate'`**：<br>• `halfUp` / `round` 四舍五入<br>• `banker` / `halfEven` 银行家（五成双）<br>• `ceil` / `floor` 朝 `+∞` / `-∞`（同 `Math`）<br>• `expand` 远离 0<br>详见[下方](#舍入-rounding) |
| `thousands` | `boolean \| 'us' \| 'eu' \| 'in'` | 千分位分组（默认不分组）：<br>• `true` = 美式 `1,234.50`<br>• `'eu'` = 欧式 `1.234,50`<br>• `'in'` = 印度式 `1,23,456` |
| `compact` | `boolean \| 'zh'` | 大数缩写（默认不缩写）：<br>• `true` = K/M/B/T（`1.23M`）<br>• `'zh'` = 万 / 亿（`123.45万`）|
| `clamp` | `[min, max]` | 把结果钳制进区间：<br>小于 `min` 取 `min`、大于 `max` 取 `max`<br>（在格式化**之前**生效）|
| `output` | `'percent' \| 'fraction' \| 'scientific' \| 'number'`<br>（或符号 `%%` `//` `e` `num`）| 输出形态，**互斥只生效一个**：<br>• `percent` 百分比（×100 加 %）<br>• `fraction` 最简分数<br>• `scientific` 科学记数<br>• `number` 返回 number（其余返回 string）|
| `plus` | `boolean` | `true` 时正数前面加 `+`<br>（负数和 0 不加）|
| `pad` | `number` | 整数左侧补零到 N 位<br>（`5` 配 `pad: 3` → `"005"`，不影响小数）|


## 小数位 `decimals`

数字 = 固定位（不足补 0）；对象 = 区间（`max` 最多、`min` 最少）：

```ts twoslash
import { fmt } from '@wzo/calc'
// ---cut---
fmt(0.1234, { decimals: 2 }) // "0.12"   固定 2 位
fmt(1.1, { decimals: { max: 2 } }) // "1.1"   最多 2 位，不补 0
fmt(1, { decimals: { min: 2 } }) // "1.00"  最少 2 位
fmt(1.2345, { decimals: { min: 1, max: 3 } }) // "1.234"
```

## 舍入 `rounding`

默认 **截断**（向零，不丢精度）。每种策略都有一个 JS 风格别名，记不住正式名就用 JS 的：

| 值 | JS 别名 | 含义 |
| :--- | :--- | :--- |
| `'truncate'` | `'trunc'` | 向零截断（同 `Math.trunc`），默认 |
| `'halfUp'` | `'round'` | 四舍五入（同 `Math.round`）|
| `'banker'` | `'halfEven'` | 银行家：四舍六入五成双（同 `Intl.NumberFormat` 的 `roundingMode: 'halfEven'`）|
| `'ceil'` | — | 朝 `+∞` 取整（同 `Math.ceil`）|
| `'floor'` | — | 朝 `-∞` 取整（同 `Math.floor`）|
| `'expand'` | — | 远离 0 取整 |

三种定向取整只在**负数**时有区别（`-1.1` → `ceil` 得 `-1`、`floor` 得 `-2`、`expand` 得 `-2`）；正数时都等同向上进位。

```ts twoslash
import { fmt } from '@wzo/calc'
// ---cut---
fmt(1.235, { decimals: 2 }) // "1.23"  默认截断
fmt(1.235, { decimals: 2, rounding: 'round' }) // "1.24"  四舍五入
fmt(2.5, { decimals: 0, rounding: 'halfEven' }) // "2"     银行家：取偶
fmt(3.5, { decimals: 0, rounding: 'halfEven' }) // "4"     取偶
fmt(-1.001, { decimals: 0, rounding: 'ceil' }) // "-1"   朝 +∞
fmt(-1.001, { decimals: 0, rounding: 'floor' }) // "-2"  朝 -∞
fmt(-1.001, { decimals: 0, rounding: 'expand' }) // "-2" 远离 0
```

::callout{type="info"}
注意：JS 的 `Number.prototype.toFixed` **不是**银行家舍入（它近似四舍五入，且有浮点 bug，如 `(1.005).toFixed(2)` → `"1.00"`）。要银行家舍入用 `'banker'` / `'halfEven'`。
::

## 千分位 `thousands`

`true` = 美式；或指定地区预设：

```ts twoslash
import { fmt } from '@wzo/calc'
// ---cut---
fmt(1234.5, { decimals: 2, thousands: true }) // "1,234.50"（美式）
fmt(1234.5, { decimals: 2, thousands: 'eu' }) // "1.234,50"（欧式）
fmt(123456, { thousands: 'in' }) // "1,23,456" （印度式）
```

## 紧凑 `compact`

`true` = K/M/B/T；或 `'zh'` 中文单位：

```ts twoslash
import { fmt } from '@wzo/calc'
// ---cut---
fmt(1234567, { decimals: 2, compact: true }) // "1.23M"
fmt(1234567, { decimals: 2, compact: 'zh' }) // "123.45万"
```

## 输出形态 `output`

互斥的输出形态，**可读词或 token 符号都行**：

```ts twoslash
import { fmt } from '@wzo/calc'
// ---cut---
fmt(0.5, { output: 'percent' }) // "50%"   （也可写 '%%'）
fmt(0.5, { output: 'fraction' }) // "1/2"   （也可写 '//'）
fmt(1000, { output: 'scientific' }) // "1e+3"  （也可写 'e'）
fmt(0.1234, { output: 'number', decimals: 2 }) // 0.12 ← number 类型（也可写 'num'）
```

## 范围限制 `clamp`

把结果钳制到 `[min, max]`，小于下界取下界、大于上界取上界，可与其他字段组合：

```ts twoslash
import { calc } from '@wzo/calc'
// ---cut---
calc('150', { _fmt: { clamp: [0, 100] } }) // "100"
calc('-50', { _fmt: { clamp: [0, 100] } }) // "0"
calc('150.567', { _fmt: { clamp: [0, 100], decimals: 2 } }) // "100.00"
```

## 正号 / 补零

```ts twoslash
import { fmt } from '@wzo/calc'
// ---cut---
fmt(1, { plus: true }) // "+1"（负数与 0 不加）
fmt(5, { pad: 3 }) // "005"
```

## 组合用法

字段可自由组合：

```ts twoslash
import { fmt } from '@wzo/calc'
// ---cut---
fmt(1234.5, { decimals: 2, thousands: true, rounding: 'round', plus: true }) // "+1,234.50"
fmt(12345678, { compact: 'zh', decimals: { max: 2 } }) // "1234.56万"
```
