# API 总览

`@wzo/calc` 的全部导出，按用途分组：

## 表达式 / 格式化

- [`calc(expr, options?)`](/api/calc) —— 计算用：表达式求值 +（可选）格式化，**出错抛异常**
- [`fmt(value, options?)`](/api/fmt) —— 展示用：calc 的展示版（支持运算 + 格式化），**出错走 `_error` 兜底**

## 链式运算

- [`chainAdd / chainSub / chainMul / chainDiv`](/api/chain) —— 链式四则运算，`chainAdd(10).sub(3).mul(2)()`

## 独立函数

- [`add / sub / mul / div / sqrt / pow / mod`](/api/standalone) —— 返回 `number`
- [`addStr / … / sqrtStr / powStr / modStr`](/api/standalone) —— 返回高精度 `string`

## 聚合

- [`calcSum / calcAvg / calcMedian / calcMax / calcMin`](/api/aggregate) —— 数组 / 对象数组聚合

## 全局配置

- [`setConfig / resetConfig / getConfig`](/api/config) —— 全局：错误兜底、默认 format、除法精度
- 单次精度：`div` / `divStr` / `chainDiv` / `calcAvg` 末尾可传 [`{ _precision }`](/api/config) 只覆盖本次
