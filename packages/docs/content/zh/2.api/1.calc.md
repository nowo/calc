# `calc(expr, options?)`

主入口函数。把表达式字符串解析、求值、（可选）格式化，返回字符串或数字。

::callout{type="warning"}
**`calc` 面向计算：出错（非法表达式、未知函数等）直接抛异常**，由你自己 try/catch。需要出错兜底的**展示**场景请用 [`fmt`](/api/fmt)——它用法一致、同样支持运算，但出错走 `_error` 兜底。**计算用 `calc`，展示用 `fmt`。**
::

## 签名

```ts
calc(expr: string, options?: ICalcOptions): string | number

interface ICalcOptions {
    // true ⇒ 表达式里的 % 当成单位标记（0.1% 还是 0.1，输出加 %）
    // false（默认）⇒ % 当成除以 100
    _unit?: boolean

    // 显式 format：IFormat 选项对象
    _fmt?: IFormat

    // 除法精度（默认取全局 _precision = 50）
    _precision?: number

    // 逐步求值调试：true ⇒ 打印 console；传函数 ⇒ 用回调接收 IDebugInfo
    _debug?: boolean | ((info: IDebugInfo) => void)
}
```

> 出错即抛，没有 `_error` 选项。要兜底就用 [`fmt`](/api/fmt)。

## 在线试一试

下面的代码可以直接编辑并运行（`⌘/Ctrl + Enter` 或点「运行」），`calc`、`fmt`、`chainAdd` 等所有导出已注入作用域：

::calc-playground

## 用法

### 基本算术

```ts twoslash
import { calc } from '@wzo/calc'
// ---cut---
calc('0.1 + 0.2') // "0.3"
calc('(1 + 2) * 3') // "9"
calc('-5 + 3') // "-2"
```

### 用模板插值代替变量

`calc` **不支持变量**——把值直接写进表达式（模板插值）即可，值可见、可调试：

```ts twoslash
import { calc } from '@wzo/calc'
// ---cut---
const a = 1
const b = 2
calc(`${a} + ${b}`) // "3"

const price = 99.99
calc(`${price} * 1.07`) // "106.9893"
```

::callout{type="info"}
表达式是**纯算术**：只有四则运算、括号、数学函数。**条件判断请在 JS 外层写**，而不是塞进表达式：

```ts twoslash
import { calc } from '@wzo/calc'
declare const a: number
// ---cut---
// 满 100 打 9 折
const final = a > 100 ? calc(`${a} * 0.9`) : String(a)
```

需要条件式取值时也可以用 `clamp` / `min` / `max` 等函数。
::

### 内置数学函数

表达式里可以用一组数学函数，全部用 BigInt 精度原语**精确**计算：

| 函数 | 说明 |
| :--- | :--- |
| `abs(x)` | 绝对值 |
| `sign(x)` | 符号（-1 / 0 / 1）|
| `floor(x)` `ceil(x)` | 向下 / 向上取整（同 `Math.floor`/`Math.ceil`，含负数）|
| `round(x)` `trunc(x)` | 四舍五入 / 向零截断（同 `Math.round`/`Math.trunc`）|
| `pow(x, n)` | 幂（`n` 须为整数，负数走除法）|
| `mod(a, b)` | 取模（余数符号同被除数）|
| `min(...)` `max(...)` | 最小 / 最大（变参）|
| `clamp(x, lo, hi)` | 钳制到 `[lo, hi]` |

```ts twoslash
import { calc } from '@wzo/calc'
// ---cut---
calc('pow(2, 10)') // "1024"
calc('mod(10, 3)') // "1"
calc('floor(-2.7)') // "-3"
calc('max(3, 5) * 2') // "10"
calc('clamp(150, 0, 100)') // "100"
```

::callout{type="info"}
无理数运算（`sqrt` / `sin` / `log` 等）因 BigInt 有理数模型无法精确表示，暂未提供。
::

::calc-playground
```ts
calc('0.1 + 0.2 + 0.3') // "0.6"
calc('max(3, 5) * 2') // "10"
calc('clamp(150, 0, 100)') // "100"
```
::

### 格式化（`_fmt`）

用 `_fmt` 传一个 [`IFormat` 对象](/guide/format-tokens)对结果格式化：

```ts twoslash
import { calc } from '@wzo/calc'
// ---cut---
const price = 9.9
const qty = 3
calc(`${price} * ${qty}`, { _fmt: { decimals: 2 } }) // "29.70"
calc('1234567', { _fmt: { decimals: 2, thousands: true } }) // "1,234,567.00"
```

### 出错抛异常（计算用）

未知函数、除以零、非法表达式等**直接抛异常**——`calc` 面向计算，要让错误可见：

```ts twoslash
import { calc, fmt } from '@wzo/calc'
// ---cut---
calc('1 + ') // ❌ 抛错：非法表达式
calc('1 / 0') // ❌ 抛错：除以零

// 需要兜底（展示）就用 fmt：
fmt('1 + ') // "-"（fmt 出错兜底）
fmt('1 / 0', { _error: 0 }) // 0
```

### 单位模式

```ts twoslash
import { calc } from '@wzo/calc'
// ---cut---
calc('0.1% + 0.2%', { _unit: true }) // "0.3%"

// 不开 _unit 时，% 就是除以 100
calc('0.1% + 0.2%') // "0.003"
```

### 调试 `_debug`

`_debug: true` 会打印逐步求值过程（每一步运算），方便排查表达式：

```ts twoslash
import { calc } from '@wzo/calc'
// ---cut---
calc('1 + 3 * 2', { _debug: true })
// [calc] 1 + 3 * 2
//   · 3 * 2 = 6
//   · 1 + 6 = 7
//   = 7
```

不想打印到 console，给 `_debug` 传一个函数即可拿到结构化信息（`{ expr, steps, value, result }`）：

```ts twoslash
import { calc } from '@wzo/calc'
declare const log: (x: unknown) => void
// ---cut---
calc('1 + 2', { _debug: info => log(info) })
```
