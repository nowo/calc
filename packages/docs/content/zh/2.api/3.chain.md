# Chain API

链式运算：`chainAdd / chainSub / chainMul / chainDiv` 是起点，返回一个可继续 `.add().sub().mul().div()` 的对象。
用空括号 `()` 或传一个 [`IFormat` 对象](/guide/format-tokens) 结束，返回最终值。

## 用法

```ts twoslash
import { chainAdd } from '@wzo/calc'
// ---cut---
chainAdd(1, 2, 3)() // "6"
chainAdd(10).sub(3).mul(2)() // "14"
chainAdd(100, 200).mul(2)({ decimals: 2, thousands: true }) // "600.00"
chainAdd(1000, 2000)({ decimals: 2, thousands: true }) // "3,000.00"
```

## 签名

```ts
chainAdd(...args: (string | number | bigint)[]): IChain
chainSub(...args): IChain
chainMul(...args): IChain
chainDiv(...args): IChain

interface IChain {
    add(...args): IChain
    sub(...args): IChain
    mul(...args): IChain
    div(...args): IChain
    (format?: IFormat): string | number  // 终止
}
```

起点函数的多参数会被 reduce：

- `chainAdd(a, b, c)` ≡ `a + b + c`
- `chainSub(a, b, c)` ≡ `a - b - c`
- `chainMul(a, b, c)` ≡ `a * b * c`
- `chainDiv(a, b, c)` ≡ `a / b / c`

## 跟 calc 的区别

- **calc** 用字符串表达式，灵活但有解析开销
- **chain** 直接调函数，更短小、性能更好

```ts twoslash
import { calc, chainAdd } from '@wzo/calc'
// ---cut---
// 等价
calc('(10 - 3) * 2') // "14"
chainAdd(10).sub(3).mul(2)() // "14"
```
