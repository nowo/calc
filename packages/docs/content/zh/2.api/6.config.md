# 全局配置

```ts
import { getConfig, resetConfig, setConfig } from '@wzo/calc'

interface IGlobalConfig {
    _error: string | number // fmt 出错兜底值，默认 '-'（仅 fmt 用；calc 等出错抛）
    _fmt?: IFormat // 全局默认 format（IFormat 对象），默认无
    _compact_default: string // 默认 compact preset，默认 ''
    _precision: number // 除法精度，默认 50
}
```

## 用法

```ts
// fmt 出错时返回 0 而不是 '-'（_error 只影响 fmt）
setConfig({ _error: 0 })
fmt('a + b') // 0（b 未定义）

// 全局默认带 2 位小数 + 千分位
setConfig({ _fmt: { decimals: 2, thousands: true } })
calc('1000 + 234') // "1,234.00"

// 调高除法精度
setConfig({ _precision: 100 })
divStr('1', '3') // 100 位的 0.333...

// 一键重置
resetConfig()
```

## 单次调用覆盖

局部 options 优先级高于全局，无需改全局即可只覆盖本次：

- `fmt` 的 `_error`、`calc` / `fmt` 的 `_fmt` / `_precision`
- 除法 / 平均：`div` / `divStr` / `chainDiv` / `calcAvg` 在**末尾**传 `{ _precision }`，只覆盖本次精度（不污染全局）

```ts
setConfig({ _error: '-' })
fmt('1/0') // "-"
fmt('1/0', { _error: 'ERR' }) // "ERR"  ← 单次覆盖

// 除法 / 平均：末尾 { _precision } 单次指定精度
divStr('100', '3', { _precision: 5 }) // "33.33333"
div(100, 3, { _precision: 2 }) // 33.33
calcAvg([10, 20, 25], { _precision: 2 }) // "18.33"
chainDiv(100, 3, { _precision: 5 })() // "33.33333"  ← 整条链的除法精度
```

## 项目级配置（`setConfig`）

整个应用用同一套配置——在入口（`main.ts`）**最早**执行一次：

```ts [main.ts]
import { setConfig } from '@wzo/calc'
setConfig({ _error: 0, _fmt: { decimals: 2, thousands: true } })
```

之后任何地方 `import { fmt } from '@wzo/calc'` 都用这套。

::callout{type="warning"}
`setConfig` 是**全局单例**：必须在任何 `calc` / `fmt` 调用**之前**执行，否则那次调用用的还是默认值。若只是想覆盖某次除法/平均的精度，用上面的单次 `{ _precision }` 即可，无需动全局。
::
